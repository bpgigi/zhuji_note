## AI 功能（DeepSeek v4-flash 流式）

| 功能 | system prompt 作用 | 截图 |
|------|-------------------|------|
| 总结 | 提炼笔记要点 | s08-s09 |
| 续写 | 基于上文智能续写 | s10 |
| 润色 | 优化文笔表达 | s11 |

| 关键词 | 提取核心关键词 | s13 |
| 起标题 | 自动生成标题 | s27 |

AI 特性：
- 流式逐 token 输出（SSE），「思考中…」与正文分区显示
- reasoning_content（推理过程）与 content（答案）分离渲染
- 余额查询、模型列表刷新、API Key 加密存储

## 写作辅助功能

| 功能 | 说明 |
|------|------|
| 番茄钟 | 专注写作计时，工作/休息循环 |
| 写作目标 | 每日字数目标 + 进度环 + 连续写作天数（streak） |

| 统计图表 | Canvas 自绘字数趋势 + 笔记数统计 |
